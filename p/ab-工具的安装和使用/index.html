ab 工具的安装和使用

2024-03-26 10:13:06
阅读时长: 1 分钟
阅读量：-

# 简介

ab 测试工具是 apache 旗下的测试工具

# 安装

# CentOS 

1

yum -y install httpd-tools

# Ubuntu / Debian 

1

apt -y install apache2-utils

# 使用说明

# 查看帮助 

1

ab -h

# get请求 

1
2

# url 最后需要以 / 结尾，否则会提示 url 错误
ab http://www.baidu.com/

# post 请求（携带 json 数据）

示例 json 数据如下（json 数据存储于 message.json 文件中）：

1
2
3
4

{
    "name":"zhangsan",
    "age":20
}

1
2

# 发送请求
ab -p message.json -T 'application/json' http://www.baidu.com/

# 性能测试 

1
2
3
4
5

# 设置请求数量
ab -n 100 http://www.baidu.com/

# 设置请求并发数
ab -c 50 http://www.baidu.com/

# abs

用于测试 https 请求，用法同上，命令 ab 替换为 abs
