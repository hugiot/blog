# 简介

crontab 命令常见于 Unix 和类 Unix 的操作系统之中，用于设置周期性被执行的指令。

# Example of job definition:
# .---------------- minute (0 - 59)
# |  .------------- hour (0 - 23)
# |  |  .---------- day of month (1 - 31)
# |  |  |  .------- month (1 - 12) OR jan,feb,mar,apr ...
# |  |  |  |  .---- day of week (0 - 6) (Sunday=0 or 7) OR sun,mon,tue,wed,thu,fri,sat
# |  |  |  |  |
# *  *  *  *  * user-name  command to be executed

# 格式说明

允许使用的修饰符共四种： ,，-，*，/

  • , 指定该位置列表值，例如1,2,3,4,5
  • - 指定该位置范围值，例如1-5表示1,2,3,4,5
  • * 代表该位置所有值，例如minute位为*代表每分钟执行
  • / 代表该位置频率，相当于除法，例如minute位*/10表示每十分钟，hour位*/4表示每四小时

# 最佳实践

定时任务设置建议放在 /etc/crontab 文件中，方便集中管理和查看
